## 5. Prototype Interaction Specifications

### 5.1 Micro-interactions

```yaml
Button Press:
  - Scale: 0.98 on :active
  - Duration: var(--hw-duration-fast)
  - Easing: var(--hw-ease-out)

Checkbox Toggle:
  - Check appears with scale from 0 to 1
  - Duration: var(--hw-duration-fast)
  - Easing: var(--hw-ease-spring)

Dropdown Open:
  - Slides down from 0 height
  - Opacity 0 to 1
  - Duration: var(--hw-duration-normal)
  - Easing: var(--hw-ease-out)

Modal Open:
  - Backdrop fades in
  - Modal slides up and fades in
  - Duration: var(--hw-duration-slow)
  - Easing: var(--hw-ease-out)

Toast Notification:
  - Slides in from right
  - Auto-dismisses after 5 seconds
  - Progress bar at bottom
  - Duration: var(--hw-duration-normal)

Loading Skeleton:
  - Shimmer effect moving left to right
  - Gradient animation
  - Duration: 1.5s
  - Loop: infinite
```

### 5.2 Gesture Support (Touch)

```yaml
Swipe Left on Row (Mobile):
  - Reveals quick actions (Edit, Delete)
  - Threshold: 50px
  - Snap back if not completed

Swipe Right on Row (Mobile):
  - Reveals primary action (e.g., Complete)
  - Threshold: 50px

Pull to Refresh:
  - Trigger: Pull down 80px at top of list
  - Animation: Spinner appears
  - Haptic feedback on trigger

Pinch to Zoom (Gallery):
  - Applies to image previews
  - Min zoom: 1x
  - Max zoom: 3x

Long Press:
  - Activates selection mode
  - Duration: 500ms
  - Haptic feedback
```
