| Block | Purpose |
|-------|---------|
| **Start** | Trigger the flow (record event, schedule, API) |
| **Condition** | Branch based on data values |
| **Get Approval** | Request approval from user/group |
| **Set Values** | Update record fields |
| **Create Item** | Create new record |
| **Send Notification** | Email, SMS, in-app |
| **Call API** | External integration |
| **Run Script** | Custom logic |
| **Wait** | Pause for time/event |
| **Loop** | Repeat actions |
| **End** | Complete the flow |
